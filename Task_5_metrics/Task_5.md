## Постановка задачи

Эксперт нашел 20 релевантных документов. Система нашла 4 документа в следующей последовательности релевантных и нерелевантных документов: 

**RNRNNRRNNNN**

Вычислить MAP.

## Решение


| Relevancy | Precision |
|-----------|-----------|
| **R**     | $1/1$     |
| **N**     | 0         |
| **R**     | $2/3$     |
| **N**     | 0         |
| **N**     | 0         |
| **R**     | $3/6$     |
| **R**     | $4/7$     |
| **N**     | 0         |
| ...       | 0         |

$MAP = \frac{\frac{1}{1} + \frac{2}{3} + \frac{3}{6} + \frac{4}{7}}{20} \approx 0.1369$
